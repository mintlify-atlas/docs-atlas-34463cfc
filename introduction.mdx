## Welcome to Dev-Tree Backend

Dev-Tree Backend is a modern Express.js API service built with TypeScript, providing a robust foundation for user authentication and profile management. The service features handle-based user routing, JWT authentication, and seamless integration with MongoDB and Cloudinary for data persistence and image uploads.

## Quick Overview

Dev-Tree Backend provides RESTful endpoints for:

- **User Registration & Authentication** - Create accounts and manage sessions with JWT tokens
- **Profile Management** - Update user profiles with handles, descriptions, and social links
- **Handle Search** - Validate handle availability before registration
- **Image Uploads** - Store and serve user profile images via Cloudinary

## Architecture

The service is structured around Express.js with TypeScript, featuring:

- **Controllers** - Business logic for authentication and user management
- **Models** - MongoDB/Mongoose schemas for data persistence
- **Middlewares** - JWT authentication and request validation
- **Routes** - RESTful API endpoints with express-validator
- **Configuration** - Centralized config for database, CORS, and cloud services
